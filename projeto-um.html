Cabeçalhos

Em HTML (Hypertext Markup Language), os cabeçalhos são elementos usados para estruturar o conteúdo da página, indicando a importância e hierarquia das diferentes partes do texto. Eles variam de h1 a h6, sendo h1 o mais importante e h6 o menos importante.

Links

Em HTML, os links são criados usando a tag a (anchor), permitindo que os usuários naveguem de uma página para outra ou a diferentes seções da mesma página.

Imagens

Em HTML, as imagens são incorporadas usando a tag img.
